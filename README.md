# awesome-robotics-resources

A curated list of notes and links regarding Vision-Language-Action (VLA) models, alongside local telemetry validation scripts.

## Supported Architectures
* **OpenVLA**: Multi-modal robotic controller.
* **ORB-SLAM3**: Visual SLAM library for CPU tracking.
* **RTAB-Map**: Real-time appearance-based mapping.
* **Kimera-VIO**: Metric-semantic SLAM system suitable for embedded CPUs.
